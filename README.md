# Boginfactory Landing Page

A professional, responsive landing page for Boginfactory Consulting - featuring strategic advisory services, mobile app development for iOS, and web development expertise.

## Features

- **Responsive Design** — Works seamlessly on desktop, tablet, and mobile devices
- **Modern Aesthetics** — Clean, professional navy/blue color scheme
- **Multi-Page Site** — Landing page, About Us, and Contact Us sections
- **Contact Form** — Secure email form powered by Formspree (no email exposure)
- **Professional Content** — Comprehensive background and service offerings

## Pages

- `index.html` — Landing page with service overview
- `about.html` — Detailed background and professional experience
- `contact.html` — Contact form and consultation information
- `SETUP_GUIDE.txt` — Instructions for setup and customization

## Quick Start

1. Clone this repository
2. Open `index.html` in your web browser to preview
3. Follow `SETUP_GUIDE.txt` for Formspree setup and deployment
